📌 Para que el manual sea permanente

Render borra los archivos subidos al reiniciar. Para que un manual quede permanente, súbelo también a GitHub:

1. Copia el PDF a proyecto_interlocks\manuals\
2. Copia el JSON generado a data\pages\
3. Ejecuta en CMD:
git add .
git commit -m "Nuevo manual: nombre"
git push

Render redeploya automáticamente y el manual queda disponible para siempre.
